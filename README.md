# Alerting Playground

Grafana Alloy setup with Fleet Management configuration.

## Quick Start

1. Configure environment variables in `.env`
2. Run: `docker-compose up`
3. Access Alloy UI: http://localhost:12345

## Configuration

- `config.alloy` - Alloy remote configuration
- Create the `.env` - Environment variables (credentials, URLs)
- `docker-compose.yml` - Container setup
